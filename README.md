
# Youth Climate Network

Welcome to the **Youth Climate Network** (YCN) project. YCN is a youth-led organization empowering young people to take action against climate change in Bangladesh and beyond.

## 🚀 Features & Functions

This application is built with a modern, dynamic, and premium architecture, offering a wide range of powerful features:

- **Bilingual Support (English/Bengali):** Fully localized interface allowing users to seamlessly toggle between English and Bengali for accessibility across different demographics.
- **Dynamic SEO:** Integrated `react-helmet-async` for automatic and dynamic metadata rendering (Titles, Descriptions, Open Graph, and Twitter Cards) for each page to enhance search engine visibility.
- **Premium User Interface:** Utilizes modern design aesthetics, micro-animations (Framer Motion), skeleton loaders for data fetching, and an incredibly sleek, fully animated splash screen.
- **Comprehensive Admin Panel:** A secure and dynamic admin dashboard to manage all website content.
- **Media & Gallery Management:** A fully functional media section integrated with the admin panel. Admins can easily upload, manage, and delete images, powered by Cloudinary and MongoDB.
- **Dynamic Content Sections:** Real-time updates for Events, News, Impacts, and statistics.
- **Donation & Involvement Portals:** Custom-built sections for users to easily donate, volunteer, or partner with the organization.

## 🛠️ Technology Stack

- **Frontend:** React (Vite), TypeScript, Tailwind CSS, Framer Motion, Radix UI.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB (Mongoose) for secure and scalable data storage.
- **Storage:** Cloudinary for robust media and image hosting.

## ⚙️ Running the Code

To get the project running locally:

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Start the Development Server:**
   This command starts both the frontend client and the backend server simultaneously.
   ```bash
   npm run dev
   ```
